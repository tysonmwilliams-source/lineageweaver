# Family Buildout Drafts

This directory contains draft documents for expanding the Lineageweaver family trees. Each document presents proposed genealogical data that requires user approval before becoming canon.

## Format Guide

Every piece of **new** data is marked with a checkbox:
- `[ ] YES` - Approve this data point
- `[ ] NO` - Reject this data point (alternative suggestions may follow)

## Approval Process

1. Review each document
2. Mark checkboxes with `x` for YES items: `[x] YES`
3. For NO items, check the NO box and see alternative suggestions
4. Once reviewed, the approved data can be imported into Lineageweaver
